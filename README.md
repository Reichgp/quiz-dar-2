# quiz-dar 🧠

Interactive self-assessment platform for **DAR (Desarrollo de Aplicaciones en Red)**.

This repository contains a complete question bank and assessment engine designed to help students prepare DAR-related subjects through randomized quizzes, instant feedback and detailed explanations.

Part of the DevQuiz-Hub ecosystem.

## 🛠️ Data Contract & Logic

### `dar_questions.json`

Complete question bank including:

* Questions
* Answers
* Explanations
* Metadata
* Topic classification

### `quiz.js`

Core assessment engine responsible for:

* Question rendering
* Answer validation
* Randomization
* Score calculation
* Progress management

### `index.html`

Responsive client-side application.

### `styles.css`

Visual customization and responsive layout rules.
